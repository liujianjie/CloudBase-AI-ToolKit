## 安装与查看

如果需要安装全部 CloudBase Skills，可执行：

```bash
npx skills add tencentcloudbase/cloudbase-skills
```

如果只安装当前 Skill，可执行：

```bash
npx skills add https://github.com/tencentcloudbase/skills --skill cloudrun-development
```

当前 Skill 在线查看： [cloudrun-development](https://skills.sh/tencentcloudbase/skills/cloudrun-development)
